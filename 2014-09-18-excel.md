# shortcut keys
* 日付入力
: Ctrl + ;
* 時刻入力
: Ctrl + :
* 行を非表示
: Ctrl + 9
* 行を表示
: Ctrl + Shift + 9
* 列を非表示
: Ctrl + 0
* 列を表示
: Ctrl + Shift + 0
* 数式を表示
: Ctrl + Shift + @
