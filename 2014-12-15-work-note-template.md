## About this

作業メモの書き方、テンプレートをまとめる。

## Notes

- vimrcに登録した現在時刻コマンドを作業の区切りで実行していく。

## Format,example

    # About this
    目的はB票111の調査、対策完了まで。

    - period : 10/10 定時
    - working items
        - ぐぐって調査する : 1h
        - 有識者へヒアリング : 2h
        - バグ対策 : 1h

    # Work notes
    ## ググる
    まずgoogleで「aaa 事象」などで検索。
    -> 該当なし
    [2014-12-15 10:16:12]

    Yahooで同様の調査を実施
    -> 同じく該当なし
    TODO: Yahooがつながらなかった

    => Webではこれ以上不明なため断念。
    [2014-12-15 10:26:26]

    ## 有識者へ確認
    まず(sato)へ確認。
    -> 以下内容を確認できた。
    - そもそも既存バグ。
    - マニュアルXXにそれっぽいことが書いてあった。

    まず、既存バグなので、既存B票を確認

    -> 対策済みとなっているため、対策者に確認したらデグってるだけらしい。
        -> 再度コミットしてもらう。
        TODO: XXXに修正を再コミットしてもらう

    # Memo
    自社業務として勤怠入力作業を実施 [2014-12-15 14:34:51]
    -> 完了 [2014-12-15 15:34:59]

    # Summary
    - 作業完了したが以下課題有り
        - XXXのときNulpoが発生
        - XXXのときばぐる
        - チケットに記載してクローズする
