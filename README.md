## How to install

Linux
```
curl -L https://raw.githubusercontent.com/assout/memolist/master/install.sh | bash
```

Windows (Caution: change repository git user config)
```
cd D:\admin\Documents
git clone git@github.com:assout/memolist.git
cd ./memolist

# if use cmd,
.\install.bat
# or else if use msys
cmd //c ".\install.bat"
```
